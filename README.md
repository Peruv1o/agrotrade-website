# AgroTrade Căușeni — Static Business Website

A responsive, multi-page static website built as a portfolio project. Designed for a fictional local trading business based in Căușeni, Republic of Moldova.

---

## 🌐 Live Demo

> _Not deployed yet — portfolio project_

---

## 📄 Pages

| Page | File | Description |
|------|------|-------------|
| Home | `index.html` | Hero, product overview, why us, contact info |
| About | `despre.html` | Company story and values |
| Products | `produse.html` | Full product/service catalog |
| Advantages | `avantaje.html` | Key differentiators |
| Contact | `contact.html` | Contact form and location |

---

## 🛠️ Built With

- **HTML5** — semantic structure
- **CSS3** — custom properties, grid, flexbox, animations
- **Vanilla JavaScript** — scroll reveal, interactions
- **Google Fonts** — Playfair Display + Fira Sans
- No frameworks, no dependencies, no build tools

---

## ✨ Features

- Fully responsive (mobile, tablet, desktop)
- Smooth scroll animations (Intersection Observer API)
- Animated marquee banner
- CSS custom properties for consistent theming
- Gold & forest green color palette
- SEO-ready meta tags and Open Graph on all pages
- `robots.txt` and `sitemap.xml` included
